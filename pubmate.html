PubMate

Manuscript drafting to Clarivate EndNote, without field-code stunts.

PubMate scans Microsoft Word .docx manuscripts for PMID and DOI markers, fetches reference metadata, and writes the EndNote import file and rewritten Word document needed for Cite While You Write.

Workflow

Draft with lightweight markers. Finish with real EndNote records.

Scans the manuscript

Finds explicit PMID markers, DOI labels, DOI resolver URLs, tables, and Word comments while skipping the final reference section by default.

Builds import records

Resolves PMIDs through PubMed, checks DOIs against PubMed first, and falls back to DOI metadata when a paper has no PubMed record.

Hands off to EndNote

Writes .endnote-import.enw plus an .endnote.docx containing EndNote temporary citations for Update Citations and Bibliography.

Boundary

PubMate does not fake EndNote field codes.

EndNote Cite While You Write fields are Word field-code objects owned by EndNote. PubMate uses EndNote-supported temporary citations and import files instead of trying to hand-write brittle CWYW internals.

The original input document is left untouched. PubMate writes a separate output document, import file, optional PubMed NBIB, and a JSON processing report so authors can review exactly what happened.
